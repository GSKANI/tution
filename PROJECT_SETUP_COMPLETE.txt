## 🚀 Quick Start (3 Steps)

### Step 1: Run Setup Script (Windows)
Double-click setup.bat or run:
```bash
setup.bat
```

### Step 2: Configure Environment
- Update .env (Frontend API URL)
- Update server/.env (MongoDB, Email credentials)

### Step 3: Start Development
```bash
# Terminal 1 - Frontend
npm start              # Runs on http://localhost:3000

# Terminal 2 - Backend
cd server && npm start # Runs on http://localhost:5000
```

✓ Frontend will open automatically
✓ Test form submissions
✓ Check email notifications

═══════════════════════════════════════════════════════════════════════════════

## 📋 What You Need to Configure

### 1. Frontend (.env)
```
REACT_APP_API_URL=http://localhost:5000/api
```

### 2. Backend (server/.env)
```
PORT=5000
MONGO_URI=mongodb://localhost:27017/cornerstone-mathsphere
EMAIL_USER=your-email@gmail.com
EMAIL_PASSWORD=your-app-password
ADMIN_EMAIL=admin@example.com
```

### 3. MongoDB
Options:
- Local: MongoDB Community Edition
- Cloud: MongoDB Atlas (free tier)

### 4. Email (Gmail)
1. Enable 2FA on Google Account
2. Generate App Password: https://myaccount.google.com/apppasswords
3. Use app password in EMAIL_PASSWORD

## 🔗 API Endpoints Created

POST   /api/submit-enquiry        Submit student enquiry form
POST   /api/submit-contact        Submit contact message
POST   /api/analytics              Track user events
GET    /api/admin/enquiries        Get all enquiries (admin)
PUT    /api/admin/enquiries/:id    Update enquiry status (admin)
GET    /api/health                 Health check endpoint
